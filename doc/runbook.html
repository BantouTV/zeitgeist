Background jobs

Get status of background jobs 

$ cd /var/www/webapps/zeitgeist
$ sudo bg/status

Start/restart/stop background jobs 

$ cd /var/www/webapps/zeitgeist
$ sudo bg/start all
$ sudo bg/restart all
$ sudo bg/stop all

Start/restart/stop specific job 

$ cd /var/www/webapps/zeitgeist
$ sudo bg/stop bbc_links

Be careful about stopping jobs - queues can fill up very quickly and crash rabbitmq by causing it to run out of memory.

View log files

The log files are stored in /tmp, e.g. 

$ tail -F /tmp/find_links_1.log

Website
